Sentiment classification IMDB review analysis system.

Pipeline:
1. TF-IDF
2. Logistic Regression
3. GridSearchCV

Technologies:
Python, Pandas, Scikit-Learn, Joblib

The projects shows:
1. Classification report(LR) for default(user-set) config
2. Top positive/negative features
3. Simple prediction system, the system shows review type and confidence to the review written by user
4. GridSearchCV fitting progress
5. GridSearchCV's best parameters, best score and Classification Report

The dataset was taken from Kaggle:
https://www.kaggle.com/code/nourhankarm/sentiment-analysis-of-movie-reviews-imdb-dataset/input

The results of work are stored in report.txt file.
The trained model is saved in the main folder.
